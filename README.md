## 💡 Project Overview
This project presents a structured audit of **LibreOffice**, a widely used open-source office suite.  
The aim is to understand its **origin, licensing, philosophy, and ecosystem**, along with practical implementation of Linux shell scripting.

The project also demonstrates how open-source tools can be analyzed and automated using **Bash scripts**.

---

## 🧾 About LibreOffice
LibreOffice is a free and open-source office productivity suite developed by **The Document Foundation**.  
It includes applications for:
- 📄 Word Processing (Writer)  
- 📊 Spreadsheets (Calc)  
- 📽️ Presentations (Impress)  
- 🗄️ Database Management (Base)

## ⚙️ Shell Scripts Included

### 🔹 Script 1: System Identity Report
Displays system information such as:
- Kernel version  
- Logged-in user  
- System uptime  
- Date and time  

---

### 🔹 Script 2: FOSS Package Inspector
- Checks whether LibreOffice is installed  
- Displays installation details  
- Uses conditional statements and case structure  

---

### 🔹 Script 3: Disk and Permission Auditor
- Analyzes important system directories  
- Displays permissions, ownership, and disk usage  
- Uses loops and Linux commands  

---

### 🔹 Script 4: Log File Analyzer
- Reads a file line by line  
- Counts occurrences of a keyword  
- Displays last matching lines  

---

### 🔹 Script 5: Open Source Manifesto Generator
- Takes user input  
- Generates a personalized open-source statement  
- Saves output to a text file
